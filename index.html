Códigos CSS - Tipografia

Cor

Para mudar as cores, podemos utilizar os códigos abaixo:

color: código hexadecimal da cor

color: rgba(x,x,x,x) cógigo rgb + nível de transparência

Fonte

Para alterar a fonte, podemos utilizar os códigos abaixo, sempre atentando para falsos itálicos:

font-style: italic, normal, bold

Tamanho e Altura de Linha

Para alterar tamanho e altura de linhas, podemos utilizar os códigos abaixo. Podemos colocar as medidas em px (pixels) ou em (medida relativa de pixels), que é o mais utilizado, por se ajustar às medidas do browser do usuário.

font-size: 16px

line-height: 25px

Família Tipográfica

A família tipográfica representa a fonte que será utilizada. É interessante colocar os nomes entre aspas para garantir que o sistema irá encontrar a tipografia. Caso ele não encontre, ele usa a tipografia do item seguinte.

font-family: "Helvetica", "Arial", "sans-serif"

Variante

Não é recomendado o uso da variante small caps porque força um falso small caps na tipografia selecionada. Prefira utilizar uma fonte original small caps.

font-variant: small-caps

Fonte Geral

Define diversas características em apenas uma propriedade.

A ordem seria: estilo, variante, peso, tamanho/line-height e família

font: bold, 2em, Georgia

Alinhamento

Alinha o texto à esquerda, centro, direita ou justificado.

text-align: center

text-align: justify

text-align: right

text-align: left

Transformar

Transforma o texto todo em minúsculas, maiúsculas ou toda a 1a letra maiúscula.

text-transform: uppercase

text-transform: lowercase

text-transform: capitalize

Indentar

Indenta começo de parágrafos.

text-indent: 1em

Decoração

Decora o texto com linhas, ou até mesmo pode anular a decoração de um texto.

text-decoration: underline

text-decoration: none

text-decoration: overline

text-decoration: line-through

Sombra

Adiciona uma sombra ao texto.

text-shadow: 1px, 1px, 2px, # 000

Ordem: distribuição horizontal, vertical, blur, cor.

Tracking e Quebra

Define o espaçamento entre letras e a quebra de linha

letter-spacing: 1em

white-space: nowrap

Lista

Define características para os elementos de uma lista.

Alinha as bolinhas da lista:

list-style-position: inside

Troca bolinhas da lista por outro elemento

list-style: armenian

Troca bolinhas da lista por imagens

list-style-image: url("")
